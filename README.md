🥷 Desafio Java: Ninjas de KonohaEste projeto é uma aplicação console em Java desenvolvida para praticar os fundamentos da linguagem, como variáveis, tipos de dados e estruturas condicionais. O desafio consiste em gerenciar ninjas da Vila da Folha e validar a conclusão de suas missões com base em critérios de idade e dificuldade.📋 Descrição do DesafioO objetivo é criar um programa que represente três ninjas e suas respectivas missões. Cada ninja possui:Nome (String) Idade (int) Missão (Nome, Nível de Dificuldade e Status) Regras de Negócio (Lógica de Conclusão)Para tornar o desafio mais interessante, foi aplicada a seguinte lógica para determinar se uma missão pode ser concluída:Ninjas menores de 15 anos: Só podem concluir missões de nível C ou D.Ninjas com 15 anos ou mais: Podem concluir missões de qualquer nível.🚀 Tecnologias UtilizadasJava System.out.println (para saída de dados no console) 🛠️ Como ExecutarCertifique-se de ter o JDK instalado em sua máquina.Clone este repositório.Compile o arquivo principal:Bashjavac Main.java
Execute o programa:Bashjava Main
📝 Exemplo de SaídaO programa processa a lógica e exibe o status final de cada ninja, por exemplo:Ninja: Naruto Uzumaki | Idade: 12 | Missão: Escoltar construtor (Nível C) | Status: Concluída
